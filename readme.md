## Browser Extension
The `prompt_optimizer_extension` folder contains a browser extension that allows you to use the TokenShrink API directly from your browser.

### How to install:
1. Open your browser (Chrome/Edge/Brave).
2. Navigate to `chrome://extensions/`.
3. Enable **Developer mode** (top right switch).
4. Click **Load unpacked**.
5. Select the `prompt_optimizer_extension` folder from this repository.# TokenShrink-Extension
